# Stock Trend Predictor - Setup Guide

## Zero Configuration Required

This application works immediately without any API keys or configuration!

### Data Sources

**Cryptocurrencies** - Powered by [CoinGecko API](https://www.coingecko.com/)
- Real-time prices for Bitcoin, Ethereum, and Binance Coin
- 24-hour volume and price changes
- Historical data (30 days)
- **Completely free, no API key required**

**Indian Stocks** - Realistic mock data
- Reliance Industries, TCS, HDFC Bank, Infosys, ICICI Bank
- Proper market hours simulation (9:15 AM - 3:30 PM IST)
- Realistic volatility patterns

### Automatic Fallback

When CoinGecko rate limits are hit:
- App automatically generates realistic mock data
- All features continue to work seamlessly
- No errors or interruptions

### Supported Markets

**Cryptocurrencies:**
- BTC (Bitcoin)
- ETH (Ethereum)
- BNB (Binance Coin)

**Indian Stocks:**
- RELIANCE.BSE (Reliance Industries)
- TCS.BSE (Tata Consultancy Services)
- HDFCBANK.BSE (HDFC Bank)
- INFY.BSE (Infosys)
- ICICIBANK.BSE (ICICI Bank)

### Currency

All prices are displayed in **Indian Rupees (₹)** including cryptocurrency prices.

### Features

- ✅ Real-time cryptocurrency prices from CoinGecko
- ✅ Technical indicators (EMA, RSI, MACD, Bollinger Bands, etc.)
- ✅ Rule-based trading signal predictions
- ✅ Interactive charts with candlestick visualization
- ✅ Performance tracking and backtesting
- ✅ Automatic fallback to mock data
- ✅ Beautiful dark theme with light chart backgrounds

### Testing

Simply select different tickers from the dropdown to explore:
- Live crypto data (when API is available)
- Realistic mock stock data for Indian companies
- Complete technical analysis and predictions

No setup, no API keys, no configuration - just start exploring!
